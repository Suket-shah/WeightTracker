# WeightTracker

Developed an application in Java to track, monitor and conduct
data analysis on individual weight gain/loss thereby providing users a pattern
of their health. Uses multiple ArrayLists to store data, Scanners that traverse
stored data, and class decomposition.

## Getting Started

Program runs on Java compilers such as Eclipse. 


## Running the tests

Navigate the menu and follow instructions within the program accordingly.
